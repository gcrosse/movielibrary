About Movie Library

Welcome to Movie Library! This web application lets you add, view, and manage your favorite movies. You can filter and sort movies by genre or release year, and all data is stored persistently in a database.

This project demonstrates:

  • Spring Boot & Spring Data JPA integration
  • Thymeleaf templating for dynamic web pages
  • Form validation and database persistence
  • Bootstrap styling for a responsive interface
